# skills/

Claude Code slash command skills. Each subdirectory contains a `SKILL.md` that defines the skill.

## Available Skills

### `/pepper`
Development mode. Activates Pepper's CLI tooling for interacting with the running iOS app — look, tap, scroll, inspect state, debug issues. Includes allowed commands for `pepper-ctl`, `make build`, `simctl`, etc.
